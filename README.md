# Personal Portfolio Website

## 🚀 Features
- Responsive React frontend
- Contact form with backend API
- Clean UI for showcasing projects

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js

## ▶️ Run Locally

### Frontend
cd frontend
npm install
npm start

### Backend
cd backend
npm install
node server.js

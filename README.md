# Personal Portfolio Website

## 🚀 Features
- Responsive React frontend
- Contact form with backend API
- Clean UI for showcasing projects

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js

## ▶️ Run Locally

### Frontend
cd frontend
npm install
npm start

### Backend
cd backend
npm install
node server.js

### Resume Project Generator
Run the Python script from the portfolio root to generate resume-style project summaries and scaffold project folders:

python resume_projects.py

Or generate only the Markdown summary:

python resume_projects.py --no-scaffold

## 🧩 Simple Python Projects
These example projects are included inside the portfolio and can be run locally with Python.

- `simple-todo-app` — a minimal command-line todo list manager
- `weather-fetcher` — a simple weather lookup demo script
- `text-analyzer` — analyze text for word counts and top words
- `budget-tracker` — track income and expenses with a CLI

## 🌩️ Cloud Services Practice Website
A simple static landing page built for practice with a cloud services portfolio theme.

- `cloud-portfolio/index.html` — the homepage
- `cloud-portfolio/styles.css` — styling for the practice site

Open `cloud-portfolio/index.html` in a browser to view the page.
